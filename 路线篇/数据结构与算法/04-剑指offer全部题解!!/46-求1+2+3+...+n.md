## 题目描述

    求1+2+3+...+n，要求不能使用乘除法、for、while、if、else、switch、case
    等关键字及条件判断语句（A?B:C）。
    
## 解析

    使用短路特性
    
## java
    
    public class Solution {  
        public int Sum_Solution(int n) {  
            int res = n;  
            boolean flag = (n>0)&&((res+=Sum_Solution(n-1))>0);  
            return res;  
        }  
    }
